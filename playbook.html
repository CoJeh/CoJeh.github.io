🧭 How to use this handbook

Two records work together. Don't confuse them:
📓 Learning Journal = a diary
What happened today — dated, with the story and the feelings. One entry per work day.
📘 Playbook = a recipe
What to do next time — timeless, de-personalized steps, organized by stage. This page.
You don't read the playbook daily. You add to it whenever you crack a repeatable step, and you use it when: ① building a second game, ② showing your method to an employer, ③ teaching someone else.

🚀 Launch Roadmap — from build to published & earning

The 8-step path to getting a mini-game live on WeChat (ad-only / individual entity).
✅ No game license (版号) needed
Required only for in-app-purchase games. Ad-only (IAA) skips the hardest, slowest gate.
💰 Cost is small & fixed
WeChat verification ≈ ¥300/yr; government filing is free; a server is needed only if you self-host a backend. Players cost you nothing — ads earn you money.
🗄️ No backend required
A simple casual game stores progress locally + WeChat login + WeChat's friend-leaderboard data domain. Add serverless "WeChat CloudBase" later only if you need global ranks.
